Overview

This document summarizes how species changed between the major assessment and the 2025 update for:

  • Long-term trend
  • Current trend
  • Range
  • Priority category

The analysis is grouped by the 2023 conservation priority .

Total Number of Species With Changes

Code 
overall_summary <- changes %>%
  summarise(
    LongTerm_Changes = sum(Major_LongTerm != Latest_LongTerm, na.rm = TRUE),
    Current_Changes  = sum(Major_Current != Latest_Current, na.rm = TRUE),
    Range_Changes    = sum(Major_Range != Latest_Range, na.rm = TRUE),
    Priority_Changes = sum(Major_Priority != Latest_Priority, na.rm = TRUE)
  )

Summary by Major Priority

Code 
summary_by_priority <- changes %>%
  group_by(Major_Priority) %>%
  summarise(
    Total_Species = n(),

    LongTerm_Changes =
      sum(Major_LongTerm != Latest_LongTerm, na.rm = TRUE),

    Current_Changes =
      sum(Major_Current != Latest_Current, na.rm = TRUE),

    Range_Changes =
      sum(Major_Range != Latest_Range, na.rm = TRUE),

    Priority_Changes =
      sum(Major_Priority != Latest_Priority, na.rm = TRUE)
  )
